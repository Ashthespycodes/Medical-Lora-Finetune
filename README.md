# Medical LoRA Finetune

This repository contains a Jupyter Notebook for medical-domain fine-tuning using Low-Rank Adaptation (LoRA) techniques. It is intended to document the experiment workflow and store the notebook used for training and evaluation.

## Contents

- `medical_lora_finetune (2).ipynb` - Main notebook showcasing dataset preparation, model configuration, and fine-tuning procedure.

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Ashthespycodes/Medical-Lora-Finetune.git
cd Medical-Lora-Finetune
```

2. Create and activate a Python environment:

```bash
python -m venv venv
venv\Scripts\activate
```

3. Install required packages. If the notebook includes a requirements file or dependency list, install them now:

```bash
pip install -r requirements.txt
```

4. Open the notebook in Jupyter Lab or Notebook:

```bash
jupyter lab
```
